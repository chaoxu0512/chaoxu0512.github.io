# 一、设置linux系统时间为北京时间

使用Xshell连接VPS后，在终端输入：

```bash
# 1.删除自带的localtime
rm -rf /etc/localtime
# 2.创建软链接到localtime
ln -s /usr/share/zoneinfo/Asia/Shanghai /etc/localtime
```

```bash
# 验证一下：
date
# Fri Sep 18 19:44:48 CST 2020
```
